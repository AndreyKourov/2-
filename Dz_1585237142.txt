Ссылка на репозиторий за 2 занятие: https://github.com/IIlinovn/LessonTwo

Ссылка на официальную документацию по установке: https://angular.io/guide/setup-local

Как его использовать:
 - Установить, если не установлен git.
 - Установить, если не установлен nodejs.
 - Установить, если не установлен @angular/cli.
 - На странице https://github.com/IIlinovn/LessonTwo найти ссылку на clone (https://github.com/IIlinovn/LessonTwo.git)
 - Открыть vs code, выоплнить команду git clone https://github.com/IIlinovn/LessonTwo.git
 - Выполнить команду cd LessonTwo для перехода в терминале в каталог проекта
 - Выполнить команду npm install для установки зависимостей (при  этом будет создан каталог node_modules)
 - Выполнить команду ng server --open для запуска приложения.

В качестве репозиторий рекомендуется использовать Ваш код, созданный на занятии.
